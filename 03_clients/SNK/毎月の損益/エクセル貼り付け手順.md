# SNK ダイカスト事業部 ダッシュボード更新手順

## 毎月の更新フロー

```
エクセルのデータをコピー
       ↓
ターミナルで update_dashboard.py を実行
       ↓
データを貼り付けて確認 → y を押す
       ↓
diecast_dashboard.html が自動更新
       ↓
ブラウザで再読み込み
```

## ② スクリプト実行方法

### ターミナルを開く
VS Code で `03_clients/SNK` フォルダを右クリック → 「統合ターミナルで開く」

### 実行
```bash
python update_dashboard.py
```

### 貼り付け手順
1. エクセルで対象範囲を選択してコピー（Ctrl+C）
2. ターミナルのプロンプトに貼り付け（Ctrl+V または右クリック→貼り付け）
3. 終了キー入力:
   - **Windows**: Enter → **Ctrl+Z** → Enter
   - **Mac**: **Ctrl+D**
4. 解析結果を確認して `y` → Enter

---

## ③ ブラウザで確認

`diecast_dashboard.html` をブラウザで開いて **F5（再読み込み）** するとデータが反映されます。

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| `❌ 解析エラー: データが空です` | 貼り付けができていない。Ctrl+V してから終了キーを押す |
| 数値が 0 になる | エクセルのセルに数式エラーや文字が入っていないか確認 |
| グラフが表示されない | ブラウザの再読み込み（F5）を試す |
| 月が反映されない | ヘッダー行の月名が正しいか確認（「1月」など） |
