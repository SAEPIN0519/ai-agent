# AIに聞く3つの質問 — 「うちの店の集客プラン」が完成します

> **使い方：** 〇〇の部分をあなたのお店の情報に書き換えて、FELOにそのまま貼り付けてください。
> ステップ1 → 2 → 3の順番に、**同じチャット画面のまま**続けて質問してください。
>
> **講座が終わった後も使えます：** 季節が変わったとき、新メニューを出したとき、ターゲットを変えたいとき — この3ステップをもう一度やれば、新しい集客プランが作れます。

---

## ステップ1：お客様の「1日」を見える化する

> まず、あなたの理想のお客様がどんな生活をしているかをAIに描き出してもらいます。
> **「いつ」「どこで」あなたのお店の情報が届くか** がわかります。

```
あなたは地方の小規模店舗に特化した消費者行動の専門家です。
最新の生活者調査データや消費者行動の統計を参照しながら回答してください。

私のお店の「理想のお客様」の1日と、今の集客の改善点を分析してください。

【私のお店の情報】
・業種：〇〇（例：パン屋、美容室、整体院、カフェ、ネイルサロン）
・場所：〇〇（例：岐阜県関市）
・理想のお客様：〇〇（例：30代共働きの女性、近隣に住む50代主婦、子育て中のママ）
・そのお客様の悩み：〇〇（例：夕食の準備が大変、肩こりがひどい、自分の時間がない）

【今やっている集客】
・使っているSNS：〇〇（例：Instagram、LINE公式、なし）
・Instagramの投稿頻度：〇〇（例：週1回、月2〜3回、ほぼ放置）
・Instagramで発信しているキーワードや内容：〇〇（例：新メニュー紹介、店内写真、特になし）
・SNS以外の集客（リアル）：〇〇（例：チラシ、口コミ、看板、地域イベント、特になし）

【こう出力してください】

まず、最も重要な発見を冒頭3行にまとめてください。

そのあと以下の内容を出してください：

1. 理想のお客様の1日のスケジュール（朝・昼・夕・夜の行動と気分）
2. スマホを見る「隙間時間」はいつか（通勤中、昼休み、子どもの昼寝中 など）
3. 私のお店のことを「思い出す」または「情報に触れる」可能性が高いタイミング
4. 今のInstagramの投稿内容・キーワードへの診断：
   - ターゲットに届いているか？ 改善すべき点は？
   - このターゲットに刺さるInstagramのキーワード・ハッシュタグを10個提案
   - 投稿するベストな時間帯と曜日
5. 今のリアル集客（チラシ・口コミ等）への診断：
   - うまく機能しているか？ もっと効果を出すための工夫は？
6. 同じ業種・ターゲットで集客に成功している店舗の実例があれば紹介してください
```

**FELOの回答が出たら：**
- 「スマホを見る隙間時間」をアクションシートにメモする
- 今のInstagramへの診断結果で「これは変えよう」と思ったことをメモ
- 提案されたハッシュタグで使えそうなものにチェック

## ステップ2：「知る → 気になる → 行ってみる」の流れを分析する

> 次に、お客様があなたのお店を**知ってから来店するまでの心の動き**をAIに分析してもらいます。
> **「何を」「どの方法で」伝えれば来てもらえるか** がわかります。

```
あなたは地方の小規模店舗のマーケティング戦略の専門家です。
最新の集客事例やデータを踏まえて回答してください。

先ほどのターゲットが、私のお店を「知る」→「気になる」→「行ってみよう」と思うまでの心の動きを分析してください。

【私のお店の情報】
・業種：〇〇
・場所：〇〇

【分析してほしいこと】

まず、最も重要な発見を冒頭3行にまとめてください。

そのあと以下の内容を出してください：

1. お客様が店を知るきっかけになる媒体は何か
   - SNS（Instagram、LINE、TikTokなど）、チラシ、看板、口コミ、Googleマップ、地域情報サイトなど、あらゆる可能性から最適なものを選んでください
   - なぜその媒体が最適なのか、理由もつけてください

2. 各段階でのお客様の気持ち（不安・期待・迷い）を具体的に書いてください

| 段階 | お客様の気持ち | 届けるべき情報 | 最適な媒体 |
|------|-------------|-------------|-----------|
| 知る | | | |
| 気になる | | | |
| 比べる・迷う | | | |
| 行ってみる | | | |

3. 来店を迷っているお客様の背中を押す「最後の一言」を3パターン提案してください

4. 同じ業種で、この流れをうまく作っている店舗の事例があれば紹介してください
```

**FELOの回答が出たら：**
- 表の「最適な媒体」をアクションシートにメモする
- 「最後の一言」で一番しっくり来るものに丸をつける

## ステップ3：明日からできるアクションプランを作る

> 最後に、ステップ1・2の分析を踏まえて、**お金をかけずに明日からできること**をAIに提案してもらいます。
> **「いつ」「何を」「どうやる」** が具体的にわかります。

```
あなたは小規模店舗の集客改善を専門とするコンサルタントです。
最新の成功事例を踏まえて、すぐに実行できるアクションだけを提案してください。

これまでの分析を踏まえて、広告費0円で明日からすぐ実行できる集客アクションを、**ネット（SNS・Web）**と**リアル（店頭・対面）**に分けて提案してください。

【私のお店の情報】
・業種：〇〇
・場所：〇〇
・一番の悩み：〇〇（例：平日の午後にお客さんが来ない、リピーターが増えない）

【条件】
・お金をかけない（コスト0円）方法だけ
・特別なスキルがなくても1人でできること
・ITが苦手でもできること

【こう出力してください】

まず、最も効果が高いアクションを冒頭3行にまとめてください。

そのあと、**ネットとリアルに分けて**アクションを出してください：

■ ネット施策（SNS・Web）

【ネット①：明日やること】
・やること：
・具体的な手順（3ステップ以内で）：
  ①
  ②
  ③
・投稿するベストな時間帯：（ステップ1の分析を踏まえて）
・使うべきキーワード・ハッシュタグ：
・なぜ効果があるのか：

【ネット②：1週間以内にやること】
（同じ形式で）

【ネット③：1ヶ月以内に仕組み化すること】
（同じ形式で）

■ リアル施策（店頭・対面・地域）

【リアル①：明日やること】
・やること：
・具体的な手順（3ステップ以内で）：
  ①
  ②
  ③
・なぜ効果があるのか：

【リアル②：1週間以内にやること】
（同じ形式で）

【リアル③：1ヶ月以内にやること】
（同じ形式で）

最後に、ネット施策とリアル施策を両方実行した場合の「3ヶ月後のビフォー・アフター」を見せてください：
・ビフォー（今の状態）：
・アフター（3ヶ月後）：
```

**FELOの回答が出たら：**
- ネット施策3つ・リアル施策3つをアクションシートに書き写す
- ネットとリアルそれぞれで「明日やること」に★をつける — これが今日の持ち帰り
- **グループシェアで他の人の施策も参考にする**

## 【ボーナス】ステップ4：優先順位を整理する（時間に余裕がある方向け）

> ステップ1〜3で出てきたアイデアが多すぎて迷うときに使ってください。

```
あなたは店舗経営のコーチです。
私は今日の講座で以下のアイデアが出ました。
1日に集客作業に使える時間は〇〇分（例：30分）です。
ITスキルは〇〇（例：LINEとInstagramは使える、パソコンは苦手）です。

どれから手をつけるべきか、優先順位をつけてください。

【出てきたアイデア】
・〇〇
・〇〇
・〇〇
（FELOの回答から気になったものをここに貼ってください）

以下の形式で出してください：

| 優先度 | やること | 理由 | 1日あたりの所要時間 | 効果が出るまでの目安 |
|--------|---------|------|-------------------|-------------------|

そのうえで「最初の1週間でやること」を月〜日の曜日ごとに作ってください。
```

## 講師向けガイド

### プロンプト設計のポイント（なぜこの構成なのか）

| 技術 | 理由 |
|------|------|
| 「あなたは〇〇の専門家です」（ロール設定） | FELOの回答品質が上がる |
| 「最新のデータや事例を踏まえて」 | FELOのAI検索力（Web検索＋AI統合）を最大活用 |
| 「冒頭3行にまとめてください」 | 長い回答でも、受講者が最初に重要ポイントを掴める |
| 「こう出力してください」（出力形式指定） | 回答がアクションシートにそのまま転記できる |
| 「3ステップ以内」「コスト0円」（制約条件） | 実行不可能な提案を排除する |
| 記入例を複数提示 | 受講者が「何を書けばいいか」で迷わない |

### 講座中の進め方
1. プロンプトは**紙で印刷して配布**する（スマホ画面の切り替え不要）
2. 〇〇の部分は**講師がスクリーンで実演しながら一緒に埋める**
3. 3つのステップは**同じチャット画面のまま連続で質問**する（FELOが前の回答を踏まえて答える）
4. FELOの回答が出たら**「冒頭3行を読んでください。使えそうですか？」**と声かけ
5. 入力が難しい方には**講師が口頭で「業種は何ですか？じゃあここにそれを入れましょう」**とサポート
