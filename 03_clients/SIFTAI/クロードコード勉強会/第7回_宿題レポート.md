# 企画書：PRO/PREMIUM会員 マネタイズ達成率向上プログラム

## 企画概要

PRO/PREMIUM会員57名全員がLv.1未満（月収10万円未達）、目標5万円の達成者は0名。マネタイズ成果があるのは6名（合計33,849円）にとどまる。会員のフェーズ別に阻害要因を特定し、フェーズごとの施策で「初収益」と「目標達成」の2段階を押し上げる運営戦略を設計する。

## 目的

会員のマネタイズ目標達成率を引き上げ、SHIFT AIの掲げる「年間100万人のAI人材を生み出す」ビジョンを「AI人材＝稼げる人材」として証明する。

| MVV要素 | 本企画との接続 |
|---|---|
| Mission: 日本をAI先進国に | AIスキルが収益に直結することを会員の実績で示す |
| Value: Impact driven | 学習量ではなく「成果（収益）」にフォーカスした支援設計 |
| Value: Ownership | 会員自身が目標と行動計画を持ち、自走できる状態を作る |

## ターゲット

- **対象者**: PRO会員41名 + PREMIUM会員18名（併用2名含む実人数57名）
- **現状**: 全員がLv.1未満。マネタイズ成果ありは6名（全体の11%）、残り51名は収益ゼロ
- **課題**: フェーズによって阻害要因が異なる

| フェーズ | 人数 | 阻害要因 |
|---|---|---|
| 1. 基礎学習 | 10名 | 何を売ればいいかわからない。学習と収益が結びつかない |
| 2. リサーチ・設計 | 3名 | 企画はあるが、出品・営業に踏み出せない |
| 3. 制作・構築 | 5名 | 制作中だが納品・販売までたどり着かない |
| 4. 出品・営業 | 3名 | 出品したが受注に至らない。価格設定や集客に課題 |
| 5. 収益発生・拡大 | 4名 | 初収益はあるが単価が低く、目標5万円に届かない |
| 未分類（週報未提出） | 32名 | 活動状況が把握できていない |

## Before / After

| Before | After |
|---|---|
| 57名全員がLv.1未満。目標5万円達成者ゼロ | 3ヶ月後にLv.1達成者を3名以上輩出 |
| マネタイズ成果ありは6名（11%）のみ | 成果あり会員を15名以上（26%超）に引き上げ |
| フェーズ1の会員が「何をすればいいかわからない」状態 | 全員が「次の1アクション」を明確に持っている |
| 週報未提出32名の状況が見えない | 全員の現在フェーズが把握され、適切な支援が届く |

## 内容・プログラム

### 施策1: フェーズ別アクションマップの配布

各フェーズの会員に「今やるべきこと」を1枚にまとめたアクションマップを配布する。

| フェーズ | 次の1アクション | ゴール |
|---|---|---|
| 1. 基礎学習 | 「自分のスキル棚卸しワーク」を実施し、売れるスキルを1つ特定する | フェーズ2へ移行 |
| 2. リサーチ・設計 | テスト案件を1件設計し、価格・納品物・所要時間を決める | フェーズ3へ移行 |
| 3. 制作・構築 | 制作物を1件完成させ、ポートフォリオに掲載する | フェーズ4へ移行 |
| 4. 出品・営業 | クラウドソーシングに1件応募する、またはSNSで1件告知する | フェーズ5へ移行 |
| 5. 収益発生・拡大 | 単価を上げる交渉、またはリピート案件の獲得を1件試みる | Lv.1達成 |

### 施策2: 週次マネタイズ進捗の可視化

現状の週次分析レポートにマネタイズ進捗セクションを追加する。

- フェーズ別の人数推移を毎週グラフ化
- 「今週フェーズが上がった会員」をDiscordで表彰
- 成果事例を具体的に共有（金額・案件内容・かかった時間）

### 施策3: 成功者によるミニ共有会（月1回・30分）

マネタイズ成果が出ている会員が、自分のやり方を共有する場を作る。

| 項目 | 内容 |
|---|---|
| 形式 | Zoom（30分・月1回）。FB会の前に実施 |
| 登壇者 | 成果ありの6名から毎月1〜2名 |
| 内容 | 何を売ったか、どう営業したか、いくらもらったか、つまずいたポイント |
| 狙い | フェーズ1〜3の会員に「自分にもできそう」と感じてもらう |

登壇候補:
- 清水秀明（16,000円・案件収益）
- 押久保欽一（15,000円・記事収益）
- 池田勝典（600円・初納品から積み上げ中）— 「ゼロから初収益」のストーリーとして最適

### 施策4: 未分類32名の状況把握キャンペーン

週報未提出の32名に対し、1問だけのアンケートを送る。

> **「今、一番困っていることを1つだけ教えてください」**
> - A: 何を売ればいいかわからない
> - B: 作っている途中で止まっている
> - C: 出品したが売れない
> - D: 忙しくて手が止まっている
> - E: その他（自由記述）

回答結果でフェーズを割り振り、施策1のアクションマップを個別配布する。

## 成功条件

- **定量**:
  - 3ヶ月後にマネタイズ成果あり会員を6名から15名に増加（2.5倍）
  - Lv.1（月収10万円）達成者を3名以上輩出
  - 週報提出率を現在の44%から70%に引き上げ
  - 未分類32名のうち20名以上からアンケート回答を得る
- **定性**:
  - 会員が「次に何をすればいいか」を自分で説明できる状態
  - 成功事例の共有が文化として定着し、会員同士が刺激し合う環境ができている

## 想定タイムライン

| 時期 | タスク |
|---|---|
| 1週目 | 未分類32名へのアンケート送信。フェーズ別アクションマップの作成 |
| 2週目 | アンケート回答の集計とフェーズ割り振り。アクションマップの個別配布 |
| 3週目 | 週次レポートにマネタイズ進捗セクションを追加。第1回ミニ共有会の登壇者調整 |
| 4週目（月末） | 第1回ミニ共有会の実施。月次振り返りと施策の効果測定 |
| 2ヶ月目以降 | 月次サイクルで継続。フェーズ移行数・収益合計の推移を追跡 |
