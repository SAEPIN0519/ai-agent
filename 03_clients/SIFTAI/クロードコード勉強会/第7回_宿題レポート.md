# 【提案】PRO/PREMIUM会員 マネタイズ達成率向上プログラム

## 企画概要

PRO/PREMIUM会員57名全員がLv.1未満（月収10万円未達）で、目標5万円の達成者は0名。マネタイズ成果ありは6名（合計33,849円）のみ。本企画では、3層アプローチ（即効施策・中期支援・継続基盤）で、「情報を渡す」だけでなく「一緒にやる」支援を組み合わせ、会員の行動変容を促す。

## 目的

会員のマネタイズ目標達成率を引き上げ、SHIFT AIの掲げる「年間100万人のAI人材を生み出す」ビジョンを「AI人材＝稼げる人材」として証明する。

| MVV要素 | 本企画との接続 |
|---|---|
| Mission: 日本をAI先進国に | AIスキルが収益に直結することを会員の実績で示す |
| Value: Impact driven | 学習量ではなく「成果（収益）」にフォーカスした支援設計 |
| Value: Ownership | 会員自身が目標と行動計画を持ち、自走できる状態を作る |

## ターゲット

- **全体対象**: PRO会員41名 + PREMIUM会員18名（併用2名含む実人数57名）
- **現状**: 全員がLv.1未満。マネタイズ成果ありは6名（全体の11%）、残り51名は収益ゼロ

### 在籍期間による優先度分け

| 区分 | 人数 | 終了日 | 対応方針 |
|---|---|---|---|

| 3月期PRO | 20名 | 2026/04/30 | 残り約6週間。即効施策（出品チャレンジ）で短期成果を狙う |
| 4月期PRO（未開始） | 3名 | 2027/03 | 開始後にプログラムへ合流 |

| フェーズ | 人数 | 阻害要因 |
|---|---|---|

| 2. リサーチ・設計 | 3名 | 企画はあるが、出品・営業に踏み出せない |
| 3. 制作・構築 | 5名 | 制作中だが納品・販売までたどり着かない |
| 4. 出品・営業 | 3名 | 出品したが受注に至らない。価格設定や集客に課題 |

## Before / After

| Before | After |
|---|---|
| 57名全員がLv.1未満。目標5万円達成者ゼロ | 3ヶ月後にLv.1達成者を2名以上輩出 |
| マネタイズ成果ありは6名（11%）のみ | 成果あり会員を11名以上に引き上げ |
| 「何をすればいいかわからない」で止まっている | 全員が一度は出品・応募を経験している |
| 運営が情報を渡すだけで行動に繋がらない | 「一緒にやる」仕組みで行動のきっかけを作れている |

## 内容・プログラム（3層構造）

### 第1層：即効施策（今週〜2週間以内）

#### 施策1: 48時間出品チャレンジ

全員で同じ期間に「とにかく1件出品する」イベントを実施する。仲間と一緒にやることで行動のハードルを下げる。

| 項目 | 内容 |
|---|---|
| 対象 | 全会員（特に3月期PRO 20名は退会前の最後のチャンス） |
| 形式 | 48時間の期間限定イベント。Discordで実況・質問・報告をリアルタイム共有 |
| 準備物 | 運営が用意：出品テンプレート、クラウドソーシング登録ガイド、価格設定の目安表 |
| ゴール | 期間内に「1件出品する」こと。売れなくてOK。出品した事実がゴール |
| フォロー | 出品完了者をDiscord成果報告チャンネルで紹介。未完了者には個別ヒアリング |

#### 施策2: 成果未報告者への状況把握アンケート

マネタイズ成果報告がない51名に対し、1問だけのアンケートを送り、現在のフェーズを把握する。

| 項目 | 内容 |
|---|---|
| 対象 | マネタイズ成果報告がない51名 |
| 形式 | Google Forms（1問のみ・1分で回答可能） |
| 質問 | 「今、一番困っていることを1つだけ教えてください」 |
| 選択肢 | A: 何を売ればいいかわからない / B: 作っている途中で止まっている / C: 出品したが売れない / D: 忙しくて手が止まっている / E: その他 |
| 活用 | 回答結果でフェーズを割り振り、施策3のアクションマップを個別配布 |

### 第2層：中期支援（1〜3ヶ月）

#### 施策3: フェーズ別アクションマップの配布

各フェーズの会員に「今やるべきこと」を1枚にまとめたアクションマップを配布する。各アクション完了でフェーズが1つ上がる。

| フェーズ | 次の1アクション |
|---|---|
| 1. 基礎学習 | 「自分のスキル棚卸しワーク」を実施し、売れるスキルを1つ特定する |
| 2. リサーチ・設計 | テスト案件を1件設計し、価格・納品物・所要時間を決める |
| 3. 制作・構築 | 制作物を1件完成させ、ポートフォリオに掲載する |
| 4. 出品・営業 | クラウドソーシングに1件応募する、またはSNSで1件告知する |
| 5. 収益発生・拡大 | 単価を上げる交渉、またはリピート案件の獲得を1件試みる |

#### 施策4: フェーズ2-5への個別コーチング

成果に最も近いフェーズ2-5の15名に対し、個別コーチングで「あと一押し」を支援する。

| 項目 | 内容 |
|---|---|
| 対象 | フェーズ2-5の15名（うち重点対象はフェーズ2-3の8名） |
| 形式 | 隔週15分の1on1（Zoom or Discord）。「次の2週間で何をするか」を一緒に決める |
| コーチ | 冴香 + コンシェルジュで分担（1人あたり7-8名を担当） |
| フェーズ2-3（8名） | 制作完了→出品までを伴走。「一緒に出品ページを作る」レベルまで支援 |
| フェーズ4-5（7名） | 単価アップ・リピート獲得の戦略を一緒に練る。月10万円への道筋を設計 |

### 第3層：継続基盤（継続）

#### 施策5: 成果の可視化と表彰（Discord + 週次レポート）

週報にマネタイズ成果報告セクションを追加済み。Discordに「成果報告チャンネル」も新設済み。これらの改善をベースに、週次レポートにランキングと進捗セクションを追加し、行動量と成果の両面を可視化する。

| 項目 | 内容 |
|---|---|
| 週報の成果報告【改善済み】 | 週報にマネタイズ成果報告セクションを追加。収益額・案件内容を毎週収集 |
| 成果報告チャンネル【改善済み】 | Discord上に設置済み。マネタイズ報告や活動内容を会員が自由に投稿できる場 |
| 日報提出率TOP5 | 週次レポートで毎週発表。行動量の多い会員を表彰 |
| 実践FB会参加TOP5 | 週次レポートで毎週発表。学びの場への参加を評価 |
| フェーズ進捗 | フェーズ別の人数推移を毎週グラフ化。「今週フェーズが上がった会員」を紹介 |
| 事例共有 | 成果事例を具体的に共有（金額・案件内容・所要時間） |

#### 施策6: 成功者によるミニ共有会（月1回・30分）

マネタイズ成果が出ている会員が、自分のやり方を共有する場を作る。

| 項目 | 内容 |
|---|---|
| 形式 | Zoom（30分・月1回）。実践フィードバック会またはウェビナーの前に実施 |
| 登壇者 | 成果ありの6名から毎月1〜2名 |
| 内容 | 販売商品・営業方法・収益額・つまずいたポイント |
| 狙い | フェーズ1〜3の会員に「自分にもできそう」と感じてもらう |

登壇候補:
- 清水秀明（16,000円・案件収益）
- 押久保欽一（15,000円・記事収益）
- 池田勝典（600円・初納品から積み上げ中）— 「ゼロから初収益」のストーリーとして最適

## 成功条件

- **定量**:
  - 3ヶ月後にマネタイズ成果あり会員を6名から11名以上に増加
  - Lv.1（月収10万円）達成者を2名以上輩出
  - 48時間チャレンジで出品者10名以上
  - 週報提出率を現在の44%から70%に引き上げ
- **定性**:
  - 「出品したことがある」会員が過半数になっている
  - 成功事例の共有が文化として定着し、会員同士が刺激し合う環境ができている

## 作った感想

今回の企画書は、Claude Codeと対話しながら作り上げた。最初は「情報を整理して渡せばいい」という発想で施策を並べていたが、冴香タイム（5人格の円卓議論）を回したことで「情報提供だけでは人は動かない」という根本的な問題に気づけた。

特に大きかった気づきは3つ。

- **「全員に均等に」は幻想だった** — 57名を月8時間で支援するのは構造的に無理。フェーズ2-3の8名に絞ることで「ROIが最も高い層に集中する」という判断ができた
- **時間軸の不一致に気づけた** — 3月期メンバー20名は2026/04/30に退会する。3ヶ月計画を組んでも彼らには届かない。即効施策（48時間チャレンジ）を第1層に置いたのはこの問題への回答
- **「一緒にやる」の重要性** — アクションマップを配るだけでは動かない。個別コーチングで「一緒に出品ページを作る」レベルまで伴走する設計にしたことで、行動変容の確度が上がった

Claude Codeの使い方としては、冴香タイムで5人格にツッコミを入れてもらうことで、自分一人では見落としていた視点（特にデータりやんの「57名÷月8時間＝1人8.4分」という指摘）を拾えたのが大きい。AIを「清書ツール」ではなく「壁打ち相手」として使うことで、企画の質が一段上がったと感じている。
