### No.6：第1層：48時間出品チャレンジ

**表示内容**
```
施策1：48時間出品チャレンジ

■ 対象: 全会員（特に3月期PRO 20名は退会前の最後のチャンス）

■ 形式: 48時間の期間限定イベント
       Discordで実況・質問・報告をリアルタイム共有

■ ゴール: 期間内に「1件出品する」こと
         売れなくてOK。出品した事実がゴール

■ 運営が用意するもの:
  ・出品テンプレート
  ・クラウドソーシング登録ガイド
  ・価格設定の目安表

■ フォロー: 完了者は成果報告chで紹介。未完了者には個別ヒアリング
```

**話す：**
第1層の目玉施策です。みんなで同じ48時間に「とにかく1件出品する」。売れなくていい。出品した事実がゴールです。仲間と一緒にやることで心理的ハードルを下げます。特に3月期の20名は4月末に退会するので、このチャレンジが最後のチャンスになります。

### No.7：第1層：状況把握アンケート

```
施策2：状況把握アンケート
```

32名の未分類会員を把握するための施策です。たった1問、1分で回答できるアンケートです。これでフェーズの割り振りができれば、次の施策3でピンポイントの支援ができます。

### No.8：第2層：フェーズ別アクションマップ

**表示内容**
```
施策3：フェーズ別アクションマップ

各フェーズの会員に「今やるべきこと」を1枚で配布

フェーズ              → 次の1アクション
──────────────────────────────────────
1. 基礎学習           → スキル棚卸しワークで売れるスキルを1つ特定
2. リサーチ・設計      → テスト案件1件の価格・納品物・所要時間を決定
3. 制作・構築          → 制作物1件を完成 → ポートフォリオに掲載
4. 出品・営業          → クラウドソーシング1件応募 or SNS1件告知
5. 収益発生・拡大      → 単価アップ交渉 or リピート獲得を1件

各アクション完了 → フェーズが1つ上がる
```

**話す：**
「何をすればいいかわからない」を解消するのがこのマップです。各フェーズの会員に「次にやるべきこと」を1つだけ提示します。1つ完了すればフェーズが上がる。シンプルに進捗を可視化できます。
