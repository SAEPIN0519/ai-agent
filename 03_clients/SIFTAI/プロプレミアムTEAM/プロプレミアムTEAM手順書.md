## 3. OKR・KPI管理ルール

MTG決定事項（2026-03-09）に基づく：

| KR | 指標 | 目標 | 備考 |
|---|---|---|---|
| KR1 | 初回面談参加率 | 100% | 面談不参加は返金対象外 |
| KR2 | 日報提出率 | 80% | 週報はKRとして深追いしない（日報80%超えれば自然に向上） |
| KR3 | 経済的インパクト | 金額ベース | 案件収益 ＋ 業務効率化の削減額を合算 |

### KR3の算出方法（メニュー表方式）
- 外注した場合の費用を基準に、AI活用による削減額を計算する
- AIタスクの標準外注価格・標準時間のリストを作成して運用
- 「5万円」目標は業務効率化の削減額も含めるとハードルが下がる

## 6. 日報・週報 回収ルール

### 回収管理の基準

| 指標 | 判定 |
|---|---|
| 日報提出率 | KR2目標：80% |
| 週報提出 | 日報80%超で自然向上を見込む |
| 週報「マネタイズ成果」欄 | 収益状況を定期把握するため必須記入 |

### 未提出者フォローの段階的対応

| タイミング | アクション | 対応者 |
|---|---|---|
| 当日夜 | Slackで日報リマインド送信 | 自動（AIエージェント） |
| 翌朝 | 未提出者に朝リマインド送信 | 自動（AIエージェント） |
| 日曜昼 | 週報リマインド送信 | 自動（AIエージェント） |
| 日曜夜 | 週報最終リマインド送信 | 自動（AIエージェント） |
| 2週連続80%以下 | DMリスト自動生成 → 担当者がDM送信 | 下記参照 |

### リマインドDMルール（2週連続80%以下）

**対象条件:**
- PROサービス対象者（DB_Pro-Members）
- 直近2週間（先々週＋先週）ともに日報提出率が80%以下（7日中5日以下）

**DM担当者の振り分け:**

| 会員のプラン | DM担当 |
|---|---|
| PRO only | 秘書チーム（門田遥・浅野・川瀬）※仲山さんは除く |
| PRO + PREMIUM | コンシェルジュ（山本大輔・安部友博） |

- 秘書チーム・コンシェルジュともにローテーションで均等に割り振り
- Discord名を表示し、DMはDiscord経由で送信
- リスト生成: `09_system/週次日報分析_discord.py` の `remind` モード（毎週月曜20:00）

### リマインド定型文

**日報リマインド（夜）:**
```
おつかれさまです！
本日の日報、まだの方はぜひ提出してください。
小さなアクションでも大丈夫です。
継続が力になります。
→ 日報・週報チャンネル: [URL]
```

**日報リマインド（翌朝）:**
```
おはようございます！
昨日の日報がまだの方は、今朝でもOKです。
振り返ってから今日をスタートしましょう。
```

**週報リマインド（日曜昼）:**
```
今週も1週間おつかれさまでした！
週報の提出はお済みですか？
今週の成果・気づき・来週の目標を記録しておきましょう。
→ 週報チャンネル: [URL]
```

**週報リマインド（日曜夜）:**

## 8. 会員アクティブ判定と非アクティブ対応

### 判定基準

| 状態 | 条件 |
|---|---|
| アクティブ | 直近2週間で日報or週報を1回以上提出 かつ FB会に1回以上参加 |
| 非アクティブ | 直近2週間で日報・週報ゼロ or FB会ゼロ |
| 要緊急フォロー | 3週間以上無反応 |

### 非アクティブ会員フォロー手順

| 段階 | タイミング | アクション | 担当 |
|---|---|---|---|
| 1 | 1〜2週間音沙汰なし | 優しい確認メッセージ | 山本大輔 |
| 2 | 継続して無反応 | 3日に1回、Discord・プレミアムルーム・メールの3チャネルでリマインダー送信 | 山本大輔 |
| 3 | 3週間以上 | 直接的なフォロー + 面談提案 | コンシェルジュ |

### フォロー定型文

**優しい確認（1〜2週間）：**
```
[名前]さん、最近どうですか？
日報や週報が見られなくて少し心配しています。
うまくいっていないことや困っていることがあれば、
いつでも個別でお話しましょう！
```

**直接的なフォロー（3週間以上）：**
```
[名前]さん、お元気ですか？
PRO活動が少し止まっているように見えて気になっています。
一度現状を整理する時間を取りませんか？
30分ほど話せる日時を教えてください。
```

### 返金保証に関する注意事項
- 面談に参加しない場合は返金対象外
- 「行動しないと返金対象外になる可能性」を示唆する文言は、大笠さんに表現を確認してから使用すること

## 9. 面談運営ルール

### 面談種別

| 種別 | 頻度 | 対象 |
|---|---|---|
| 初回面談（マンツーマン） | 加入後Day 5まで | 全会員 |
| PRO面談 | 加入後Day 7 | PRO会員 |
| 隔週進捗確認面談 | 隔週 | PREMIUM会員 |
| 特急面談 | 随時 | 要フォロー会員 |

### 面談前準備（AIエージェントが自動収集）

面談の前日までに以下を確認：

1. **直近2週間のアクティビティ**（日報提出日数・週報・FB会参加回数・案件応募数）
2. **収益状況**（現在Lv・今月の収益・目標対比・前月比）
3. **前回面談からの申し送り**（アクション達成状況）
4. **課題・詰まりポイント**（日報・週報から抽出）

### 面談アジェンダ（標準50分）

1. 近況確認（5分）
2. 数字の振り返り（10分）
3. 課題の深掘りと解決策（20分）
4. 次回までのアクション設定（10分）
5. Q&A（5分）

## 14. 週次レポート運営

### 週次日報分析レポートの項目
1. 基本統計データ（稼働時間・日報提出数TOP3）
2. 実践FB会参加分析（曜日別参加者数・参加回数TOP3）
3. 感情・傾向占有率（プラス/マイナス傾向率）
4. カテゴリー分析（行動面・心理・スキル・環境の課題分類）
5. マネタイズフェーズ × 課題分析
6. マネタイズ成果ランキング（TOP10・最上位/最下位/平均額）
7. 会員の生の声（プラス傾向・マイナス傾向）
8. 運営への提言（個別フォロー推奨・FB会改善案）
9. 非アクティブPRO会員（過去4週傾向付き）

### 個別フォロー推奨の判断基準
- スキル面でのつまずきが見られる会員 → 個別声かけ or 1on1
- フェーズ1（操作不安）の会員 → 初心者向けオンボーディング資料の充実
- 行動面の課題が多い → 応募文作成会など短時間集中ワークショップ
- マイナス傾向率が高い → 成功体験の共有会・モチベーション施策
