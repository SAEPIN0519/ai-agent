# MTG議事録

## 使い方

1. MTGの文字起こしテキストをClaudeに渡す
2. 自動で以下を抽出・整理：
   - **決定事項** — 何が決まったか
   - **アクションアイテム** — 誰が・何を・いつまでに・どうやって
3. 議事録ファイルとして保存（`YYYY-MM-DD_MTG名.md`）
4. 全メンバーの未完了タスクを `ACTION_TRACKER.md` で担当者別に一元管理

## フォルダ構成

```
MTG議事録/
├── README.md              # このファイル
├── ACTION_TRACKER.md      # 全メンバーの未完了アクション一覧（担当者別）
├── _template.md           # 議事録テンプレート
└── YYYY-MM-DD_MTG名.md   # 各回の議事録
```

## Claudeへの指示例

```
この文字起こしからMTG議事録を作成して：
（文字起こしテキストを貼り付け）
```
