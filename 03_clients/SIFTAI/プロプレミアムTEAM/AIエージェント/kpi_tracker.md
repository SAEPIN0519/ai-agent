# AIエージェント設計：KPI・アクティブ分析エージェント

## 役割
会員のKPI進捗・収益状況・アクティビティを週次で集計・分析し、
冴香さんが「今誰をフォローすべきか」を即座に判断できる情報を提供する。

## 担当業務
1. Lv.アップフォームの回答を収集・集計
2. 会員別の月次収益トレンドをグラフ化
3. アクティブ/非アクティブの自動判定
4. 「要注意」「要緊急フォロー」会員のアラート
5. 面談前資料への自動入力（meeting_prep_template.md）
6. 月次チームサマリーレポートの生成

## 判定ロジック

```
アクティブ判定:
  直近2週間で 日報1回以上提出 AND FB会1回以上参加 → アクティブ
  それ以外 → 非アクティブ

要注意判定:
  目標対比 < 80% OR アクティブ率低下傾向 → 要注意

要緊急フォロー判定:
  目標対比 < 50% OR 2週間以上無活動 → 要緊急フォロー
```

## インプット
- Lv.アップフォーム（Google Forms）の回答
- 日報・週報チャンネル投稿（アクティビティ計測）
- FB会参加状況（fb_attendance.md）
- 会員ロースター（member_roster.md）

## アウトプット
- 週次KPIダッシュボード更新（kpi_progress.md）
- 面談前資料の自動入力
- 冴香さんへのアラートSlack（毎週月曜8:00）
- 月次サマリーレポート

## 週次アウトプットイメージ

```
【週次KPIアラート - [日付]】

要緊急フォロー（2名）:
  - Aさん: 3週間無活動 / Lv.1目標未達 → 個別連絡推奨
  - Bさん: 日報2週間未提出 / 先週FB会0回

要注意（3名）:
  - Cさん: 目標対比65% → 次回面談で深掘りを

今週の成果（表彰候補）:
  - Dさん: 初めて月10万円達成！Lv.1→2 昇格

チーム全体:
  - アクティブ率: 78%（先週比 +5%）
  - 平均Lv達成率: 72%
```

- Google Sheets + Apps Script で自動集計
- Notion データベースで会員管理
- Claude API でサマリー文章生成
