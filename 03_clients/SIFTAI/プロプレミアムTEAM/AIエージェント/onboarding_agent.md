# AIエージェント設計：オンボーディング管理エージェント

## 役割
新規会員の入会初日から初回FB会参加（Day 8）まで、
ステップの完了状況を自動追跡し、詰まっているところを即座にフォローする。

## 担当業務
1. 新規会員の加入を検知し、オンボーディングチェックリストを自動生成
2. 各ステップの期限アラートを会員とコンシェルジュへ通知
3. 初回面談前の事前準備リストをLINE/Discordで自動送付
4. 7日間で完了すべき項目の進捗を毎日確認
5. 完了率レポートを冴香さんへ週次送付

## オンボーディングフロー自動化

```
Day 1 加入検知
  → ウェルカムメッセージ送信（LINE）
  → オンボーディングチェックリスト生成

Day 1（6時間後）: LINE登録確認
  → 未登録の場合: リマインド送信

Day 2: マンツーマン面談予約リマインド
  → 未予約の場合: 予約URLを再送

Day 3: Discord参加確認
  → 未参加の場合: 登録手順を再送

Day 5: マンツーマン面談 実施前日確認
  → 前日リマインド送信

Day 6: 初回アンケートURL送信
  → 面談完了確認後にトリガー

Day 7: PRO面談前の事前確認5項目送信
  → 歩き方動画・Discord動画・バンドルコース・アンケート・面談予約

Day 8〜: PROスタート質問会（毎月1日・15日）案内
```

## インプット
- 決済完了通知（会員登録情報）
- LINE登録確認
- Googleフォーム（初回アンケート）の回答
- Discord参加確認

## アウトプット
- 会員へのLINE/Discord自動メッセージ
- オンボーディング進捗シート更新（onboarding_status.md）
- 冴香さんへの日次進捗ダイジェスト
- 詰まり会員への個別フォローアラート

## 初動導線の見える化

```
加入
 └→ エルメ（自動）: ウェルカム・LINE登録・面談予約案内
     └→ LINE登録完了
         └→ エルメ（自動）: Discord案内・カレンダー登録
             └→ マンツーマン面談
                 └→ 初回アンケート送付（自動）
                     └→ PRO事前準備5項目案内
                         └→ PRO面談
                             └→ PROスタート質問会参加
                                 └→ 実践FB会スタート
```

## 実装候補ツール
- エルメ（現状使用中）の既存フローを拡張
- Googleスプレッドシートでチェックリスト管理
- Claude API でパーソナライズされたフォローメッセージ生成
