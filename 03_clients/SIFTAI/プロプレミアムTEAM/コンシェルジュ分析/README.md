# コンシェルジュ分析

プロプレミアムTEAMのコンシェルジュサービスに関する分析フォルダ。

## 目的

- コンシェルジュ対応のサービス品質にばらつきがないか検証
- 会員様からの口コミ・フィードバックを分析し改善につなげる

## データソース

- SHIFTAIのGoogle Drive（コンシェルジュ対応記録・口コミデータ）

## フォルダ構成

```
コンシェルジュ分析/
├── README.md               ← このファイル
├── サービス品質/            ← 対応品質のばらつき分析
│   └── 品質チェックシート.md
└── 口コミ分析/              ← 会員口コミ・フィードバック分析
    └── 口コミ分析テンプレート.md
```
