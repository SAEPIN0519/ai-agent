## 01　PRO初回面談とは

### 役割の定義

PRO初回面談は、新規入会した会員様の**目標を言語化し、最初の一歩を一緒に決める場**です。

先生ではなく「伴走者」。答えを教えるのではなく、会員様自身が考え、行動し、成果を出せるよう導くことが最大の役割です。

### 面談担当がやること

- 入会背景と目標のヒアリング（30〜45分）
- 短期・中期・長期の目標設定サポート
- キャリアパスの提案（AIライター/画像動画/業務効率化/SNS運用）
- プロプラン活用方法の案内（FB会・Discord・日報）
- 面談後のカルテ記入・録画保存

### 面談担当がやらないこと

- 会員様の代わりに作業をする
- 答えを一方的に教える
- 会員様の意思決定を代行する
- プラン外のサービスを約束する
- 他の会員様の個人情報・進捗を共有する

### 心構え｜3つの原則

| 原則 | 具体的な行動 |
|---|---|
| ① 主体性を引き出す | 面談担当が決めるのではなく「あなたはどう思いますか？」と会員様本人に決めさせる。自分で決めたことが行動につながる。 |
| ② 小さな成功を認める | 「できていること」を積極的に言語化して伝える。自己効力感の低下が離脱の最大原因。 |
| ③ 一人で抱え込まない | 困ったらすぐ先輩メンバーや安部に相談する。チーム連携が品質を保つ。 |

### 面談の種類と対象者

| 種類 | 対象 | タイミング | 所要時間 |
|---|---|---|---|
| 正規入会者向け初回面談 | PRO有料加入者 | 入会後 Day 7 | 約30〜45分 |
| 特別招待者向け初回面談 | 選抜された無料招待メンバー | 招待直後 | 約20〜30分 |

- **正規入会者**: 目標設定・キャリアパス提案が面談の核。じっくりヒアリングする
- **特別招待者**: サービス説明・アクセス設定が中心。守秘義務の確認が必須

## 02　業務の全体像

### 面談業務サイクル

| タイミング | やること | 所要時間目安 |
|---|---|---|
| 面談日程確定時 | Googleカレンダーに面談IDを挿入（面談内容が自動記録される） | 5分 |
| 面談前日まで | 初回アンケート・カルテ確認、準備物チェック | 10〜15分 |
| 面談当日 | 会員様との1on1面談（Google Meet） | 30〜45分 / 1名 |
| 面談直後 | カルテ記入（文字起こしURL・議事録・録画URL） | 10分 |
| 面談1週間後 | 日報・FB会参加状況の確認、必要に応じてフォロー | 5〜10分 |

### 面談スケジュールの基本ルール

- 日程調整はカレンダー予約リンクを使用（岩間さん経由で共有）
- Google Meet URLを発行し、会員様に送付。岩間さんをゲストに追加すること
- 🔴 録画・文字起こし ← **絶対に忘れない！**
- Geminiメモ（会議録自動生成）→ カルテの「初回面談文字起こし」列にURL保存

## 03　面談前準備（日程調整〜当日まで）

新規会員様が振り分けられたら、以下の5ステップで対応します。

### ❶ 日程調整

- カレンダー予約リンクを設定し、岩間さんに渡す
- 会員様が日付を指定 → カレンダーで日時を確認
- Google Meet URLを発行 → 会員様に送信
- 岩間さんをゲストに追加（内容自動反映のため）

### ❷ 面談前準備

面談前日までに以下を必ず確認しておく：

- [ ] 初回アンケートの回答内容（エルメ経由で送信済みか）
- [ ] 会員カルテ（DB_Pro-Members）の基本情報
- [ ] Discord参加状況（ロール付与済みか）
- [ ] 会員サイトへのアクセス状況
- [ ] マンツーマン面談が実施済みか（正規入会者の場合、Day 5に実施）
- [ ] 複数担当している場合は、情報を混同しないよう必ず復習する

### ❸ 初回面談当日

- 🔴 録画・文字起こし ← **絶対に忘れない！**
- 基礎情報・現在の仕事状況を確認
- 入会背景と動機をヒアリング
- 目標設定（短期・中期・長期）
- キャリアパスの提案
- プロプラン活用方法の説明
- 次のアクションを明確にしてクロージング

> 詳細な進行は [04. 面談の進め方（正規入会者）](#04-面談の進め方正規入会者) / [05. 面談の進め方（特別招待者）](#05-面談の進め方特別招待者) を参照

### ❹ 面談後の対応

- Geminiメモ（文字起こし）のURLをカルテの「初回面談文字起こし」列に記入
- 面談内容のまとめをカルテの「初回面談議事録」列に記入
- 録画URLをカルテの「初回面談録画」列に記入
- Discord・会員サイトのアクセス設定が完了していることを確認
- 必要に応じて次回面談の予約案内

### ❺ 面談1週間後フォロー

- 日報・FB会参加状況を確認
- 参加がない場合: 優しい確認メッセージを送信
- オンボーディング進捗をチェックリストで管理

## 04　面談の進め方（正規入会者）

**所要時間: 約30〜45分**

### Phase 1｜導入（3分）

> **目的:** 緊張をほぐし、面談のゴールを共有する

| 質問例 | 内容 |
|---|---|
| — | 面談者の自己紹介 |
| Q1 | 「プロプランへの加入、ありがとうございます。今日はプロプランでの目標を一緒に整理していきましょう」 |
| Q2 | 「事前アンケートの内容を拝見しましたが、もう少し詳しくお話を聞かせてください」 |

**本日の面談で扱うこと:**
- プロプランでの目標を整理する
- 現在の状況を確認する
- 目標と現状を結びつけるキャリアプランを話し合う
- 悩みや困りごとの相談

### Phase 2｜入会背景のヒアリング（5分）

> **目的:** 会員の動機・背景を理解し、適切なアドバイスの基盤を作る

| 質問例 | 内容 |
|---|---|
| Q1 | 「AIに興味を持ったきっかけは何ですか？」 |
| Q2 | 「なぜプロプランを選びましたか？」 |
| Q3 | 「今の仕事で困っていることはありますか？」 |
| Q4 | 「副業ですか？独立を目指していますか？」 |

確認すること:
- Shift AIへの入会動機（なぜAIを学ぼうと思ったか）
- プロプランを選んだ理由
- 現在の職業・ライフスタイル
- 学習に使える時間（平日/休日）

### Phase 3｜現状把握（5〜7分）

> **目的:** AI活用のレベルと学習進捗を把握する

| 質問例 | 内容 |
|---|---|
| Q1 | 「普段どんなAIツールを使っていますか？」 |
| Q2 | 「コースはどこまで進みましたか？」 |
| Q3 | 「何かアウトプットしたものはありますか？」 |
| Q4 | 「学習で詰まっているところはありますか？」 |

> **ポイント:** 面談担当がその場でメモ・整理する。この時点では評価や選別をしない。

### Phase 4｜目標設定（7〜10分）

> **目的:** 短期・中期・長期の具体的な目標を設定する

3段階で設定する:

| 期間 | 設定する内容 | よくある目標例 |
|---|---|---|
| 1ヶ月後 | すぐ取り組むこと | 基本ツール習得、初案件応募、SNS発信開始 |
| 3ヶ月後 | 中間成果 | 月3〜5万円達成、ポートフォリオ完成 |
| 1年後 | 最終目標 | 月20〜100万円、独立・副業確立 |

| 質問例 | 内容 |
|---|---|
| Q1 | 「プロプランで達成したい金額目標はありますか？」 |
| Q2 | 「いつまでに達成したいですか？」 |
| Q3 | 「どんなライフスタイルを目指していますか？」 |

> **最重要:** 金額目標は会員の口から言ってもらう（押し付けない）。「達成不可能ではない」と肯定し、具体的なステップに落とし込む。

> **時間管理のコツ:** 目標が出ない場合は「今日の仮ゴールとして○○で進めましょう」と切り替える。

### Phase 5｜キャリアパス提案（7〜10分）

> **目的:** 目標達成に向けた具体的な道筋を提案する

#### キャリアパス決定のフローチャート

```
Q: AIの経験はどのくらい？
├─ 初心者（ほぼ未経験）
│   └→ まずAIライターから（パターンA）
│       └→ 最短でマネタイズしたい → クラウドソーシング案件応募
│       └→ じっくり学びたい → SNS発信 + ポートフォリオ構築
│
├─ 少し触っている（ChatGPT程度）
│   ├─ 文章が好き → パターンA（AIライター）
│   ├─ ビジュアルに興味 → パターンB（AI画像/動画）
│   └─ 業務効率化が目的 → パターンC（社内AI推進）
│
└─ ある程度使える
    ├─ ライティング実績あり → コンサル/マーケターへ展開
    ├─ 画像/動画生成経験あり → パターンB の Step 2〜
    └─ SNS運用経験あり → パターンD（SNSマーケター）
```

#### 4つのキャリアパス

| パス | 対象 | 最初のステップ |
|---|---|---|
| 🅰 AIライター → コンサル | AI初心者、早く稼ぎたい人 | AIライティング習得 → クラウドソーシング応募 |
| 🅱 AI画像/動画 → クリエイティブ | ビジュアルに興味がある人 | Midjourney等で毎日SNS投稿 → 動画展開 |
| 🅲 業務効率化 → 社内AI推進 | 本業がある会社員 | 自身の業務でAI活用 → 削減時間を数値化 |
| 🅳 SNS運用 → マーケター | 既にSNS発信している人 | AIでコンテンツ作成効率化 → 運用代行案件 |

> **面談データ（34件分析）:** AIライター推奨が最多（53%）。初心者には迷ったらまずAIライターを提案。

### Phase 6｜プロプラン活用方法の説明（5〜7分）

> **目的:** プロプランのサービスを最大限活用してもらう

**必ず伝える3つ:**

**① 実践フィードバック会（最重要）**
- 毎日21:00〜22:00開催（日曜のみ11:00〜12:00）
- 少人数制で講師から直接フィードバックをもらえる
- リアルタイム参加が理想だが、アーカイブ視聴でもOK
- 出席レポートの提出が必要（全額返金保証の条件）

| 曜日 | テーマ | こんな人におすすめ |
|---|---|---|
| 月 | AIコンサル案件獲得実践会 | 案件獲得を目指す人 |
| 火 | 登壇練習会 | 講師・コンサルを目指す人 |
| 水 | コンサル現場ケーススタディ会 | 実践的な事例を学びたい人 |
| 木 | 案件応募・面談練習 | 応募・営業スキルを磨きたい人 |
| 金 | テスト案件添削 | 案件のクオリティを上げたい人 |
| 土 | 週間振り返り会 | 1週間の成果を整理したい人 |
| 日 | 特別会（月替わり） | 幅広く学びたい人 |

**② Discord活用**
- プロ限定コミュニティで意識の高い仲間と交流
- 何でも相談室で学習の詰まりを解消
- 日報集に毎日の活動を記録（習慣化が最重要）

**③ 日報・週報の習慣化**
- 日報: 毎日の学習・実践の記録
- 週報: 1週間の振り返り + マネタイズ成果
- 提出内容は個人カルテに自動反映

### Phase 7｜クロージング（3分）

> **目的:** 次のアクションを明確にして面談を終える

- 今日決めたアクションを復唱：「○○さん、今日決めたことはこれです！」
- Discord・FB会の参加を改めて促す
- 困ったときの相談窓口（Discord何でも相談室）を案内
- 「一緒に頑張りましょう！」で締める

> **面談後の必須アクション:** 面談で決めたアクションを、当日中にDiscordで会員様に送る。「○○さん、今日決めたことはこれです！」のひと言を添えて。

## 05　面談の進め方（特別招待者）

**所要時間: 約20〜30分**

### Phase 1｜選抜通知とお祝い（2分）

- プロプラン特別招待の説明（数万人から20名選抜）
- 日頃のDiscord活動が評価されたことを伝える
- 無料招待期間の説明

### Phase 2｜守秘義務の確認（2分）

> **必須事項:** 他の有料会員との公平性維持のため、無料招待の事実は外部に漏らさないよう依頼。会員の同意を明確に得ること。

### Phase 3｜サービス説明 + アクセス設定（10分）

- プロプランのサービス内容説明（実践FB会が最大の特典）
- Discord専用ロール付与・チャンネルアクセス確認
- 会員サイトへの登録（氏名・メールアドレス）
- 実践FB会のスケジュール確認
- 合宿アーカイブへのアクセス確認

### Phase 4｜目標設定（5〜10分）

- 既存の学習状況・キャリア目標を確認
- プロプラン期間中の具体的な目標を設定
- 日報・週報の提出習慣化を推奨

### Phase 5｜クロージング（3分）

- PROスタート質問会（毎月1日・15日）への参加案内
- 困ったときの相談先を案内
- アーカイブ活用を推奨

## 06　よくある課題と対処法

| 課題 | 対処法 |
|---|---|
| **目標が言語化できない** | 「なりたくない状態」から逆引きする。または「理想の1日」を聞く。→「毎朝何に追われていますか？」で具体化を促す |
| **インプット過多で動けない** | 「完璧でなくていい。まず手を動かして、FB会でフィードバックをもらう」。講師の方法を徹底模倣、自己流は後から |
| **何をしたいかわからない** | キャリア診断ボットの利用を案内。迷ったらまずAIライター（最短マネタイズ）を提案 |
| **目標に対して時間が取れない** | アーカイブ視聴・隙間時間活用を強調。時間制限のないSNS運用等を推奨。「料金は支払済み。アーカイブでも必ず参加して元を取って」 |
| **多スキルだが収益化できていない** | 「スキルは十分。足りないのはビジネスの導線」。一つに絞って案件獲得に集中 |
| **ネガティブ発言が続く** | 「できていること」の承認が不足している。日報リプライや面談で成功体験を積極的に言語化。自己効力感の回復が最優先 |
| **面談で時刻オーバーしそう** | Phase 2（入会背景）で5分を超えたら「今日は目標設定を中心にいきましょう」と切り替える。最も大事なのはPhase 4（目標設定）とPhase 7（クロージング）にたどり着くこと |

## 08　禁止事項・注意事項

### 禁止事項

| # | 内容 | 理由 |
|---|---|---|
| 1 | 他の会員様の個人情報・進捗を共有する | プライバシー保護 |
| 2 | プラン外のサービスや特典を独断で約束する | 運営との整合性 |
| 3 | 会員様の代わりに作業・課題を行う | 自走力の育成が目的 |
| 4 | 面談の録画・文字起こしを省略する | 記録が品質管理の基盤 |
| 5 | 返金保証の条件について独自の表現で伝える | 大笠さんに表現を確認してから伝えること |
| 6 | 問題を一人で抱え込み報告しない | 早期対応が離脱防止の鍵 |

### エスカレーション基準

以下の場合はすぐに安部に報告・相談してください。

- 会員様が退会の意思を示した場合
- ネガティブ発言が面談中に繰り返される場合
- 面談で対応しきれない技術的な質問が出た場合
- 対応に迷って判断できない場合

### 面談の質を測るKPI

| 指標 | 目標 | 測り方 |
|---|---|---|
| **アクション設定率** | **100%** | 毎回の面談で「今週やること」を必ず1つ決める。0件で終わる面談をなくす。 |
| **面談後フォロー率** | **100%** | 面談当日中にDiscordでアクション内容を送信する。 |
| **面談1週間後のアクティブ率** | **80%+** | 面談後1週間以内に日報orFB会参加が1回以上あるか確認。 |
