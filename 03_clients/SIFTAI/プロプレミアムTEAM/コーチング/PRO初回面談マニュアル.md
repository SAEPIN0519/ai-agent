## 1. 面談の種類と対象者

| 種類 | 対象 | タイミング | 所要時間 |
|---|---|---|---|
| 正規入会者向け初回面談 | PRO有料加入者 | 入会後 Day 7 | 約30〜45分 |
| 特別招待者向け初回面談 | 選抜された無料招待メンバー | 招待直後 | 約20〜30分 |

### 正規入会者と特別招待者の違い

- **正規入会者**: 目標設定・キャリアパス提案が面談の核。じっくりヒアリングする
- **特別招待者**: サービス説明・アクセス設定が中心。守秘義務の確認が必須

## 3. 面談フロー（正規入会者）

**所要時間: 約30〜45分**

### Phase 1: 導入（3分）

```
目的: 緊張をほぐし、面談のゴールを共有する
```

- 面談者の自己紹介
- 本日の面談の目的を説明:
  - プロプランでの目標を整理する
  - 現在の状況を確認する
  - 目標と現状を結びつけるキャリアプランを話し合う
  - 悩みや困りごとの相談

### Phase 2: 入会背景のヒアリング（5分）

```
目的: 会員の動機・背景を理解し、適切なアドバイスの基盤を作る
```

確認すること:
- Shift AIへの入会動機（なぜAIを学ぼうと思ったか）
- プロプランを選んだ理由
- 現在の職業・ライフスタイル
- 学習に使える時間（平日/休日）

### Phase 3: 現状把握（5〜7分）

```
目的: AI活用のレベルと学習進捗を把握する
```

確認すること:
- 現在使っているAIツール（ChatGPT, Claude, Midjourney等）
- AI活用の頻度と用途
- これまでの学習進捗（コース受講状況、アウトプット実績）
- マンツーマン面談での内容（実施済みの場合）

### Phase 4: 目標設定（7〜10分）

```
目的: 短期・中期・長期の具体的な目標を設定する
```

3段階で設定する:
| 期間 | 設定する内容 | よくある目標例 |
|---|---|---|
| 1ヶ月後 | すぐ取り組むこと | 基本ツール習得、初案件応募、SNS発信開始 |
| 3ヶ月後 | 中間成果 | 月3〜5万円達成、ポートフォリオ完成 |
| 1年後 | 最終目標 | 月20〜100万円、独立・副業確立 |

**重要ポイント:**
- 金額目標は会員の口から言ってもらう（押し付けない）
- 「達成不可能ではない」と肯定し、具体的なステップに落とし込む
- 業務効率化も「マネタイズ効果」として換算できることを伝える

### Phase 5: キャリアパス提案（7〜10分）

```
目的: 目標達成に向けた具体的なキャリアの道筋を提案する
```

→ [6. キャリアパス提案ガイド](#6-キャリアパス提案ガイド) を参照

### Phase 6: プロプラン活用方法の説明（5〜7分）

```
目的: プロプランのサービスを最大限活用してもらう
```

→ [7. プロプラン活用方法の説明ポイント](#7-プロプラン活用方法の説明ポイント) を参照

### Phase 7: クロージング（3分）

```
目的: 次のアクションを明確にして面談を終える
```

- 今日決めたアクションを復唱
- Discord・FB会の参加を改めて促す
- 困ったときの相談窓口（Discord何でも相談室）を案内
- 「一緒に頑張りましょう」で締める

## 4. 面談フロー（特別招待者）

**所要時間: 約20〜30分**

### Phase 1: 選抜通知とお祝い（2分）

- プロプラン特別招待の説明（数万人から20名選抜）
- 日頃のDiscord活動が評価されたことを伝える
- 無料招待期間の説明（〜4月末）

### Phase 2: 守秘義務の確認（2分）

**必須事項:**
- 他の有料会員との公平性維持のため、無料招待の事実は外部に漏らさないよう依頼
- 会員の同意を明確に得る

### Phase 3: サービス説明 + アクセス設定（10分）

- プロプランのサービス内容説明（実践FB会が最大の特典）
- Discord専用ロール付与・チャンネルアクセス確認
- 会員サイトへの登録（氏名・メールアドレス）
- 実践FB会のスケジュール確認
- 合宿アーカイブへのアクセス確認

### Phase 4: 目標設定（5〜10分）

- 既存の学習状況・キャリア目標を確認
- プロプラン期間中の具体的な目標を設定
- 日報・週報の提出習慣化を推奨

### Phase 5: クロージング（3分）

- プロスタート質問会（毎月1日・15日）への参加案内
- 困ったときの相談先を案内
- アーカイブ活用を推奨

## 5. ヒアリング項目と深掘り質問集

### 入会動機を掘り下げる

| 質問 | 目的 |
|---|---|
| 「AIに興味を持ったきっかけは何ですか？」 | 動機の本質を理解 |
| 「なぜプロプランを選びましたか？」 | 期待値の確認 |
| 「今の仕事で困っていることはありますか？」 | 実務課題の把握 |
| 「副業ですか？独立を目指していますか？」 | キャリアの方向性 |

### 学習状況を把握する

| 質問 | 目的 |
|---|---|
| 「普段どんなAIツールを使っていますか？」 | スキルレベルの確認 |
| 「コースはどこまで進みましたか？」 | 進捗確認 |
| 「何かアウトプットしたものはありますか？」 | 実践度の確認 |
| 「学習で詰まっているところはありますか？」 | ブロッカーの特定 |

### 目標を具体化する

| 質問 | 目的 |
|---|---|
| 「プロプランで達成したい金額目標はありますか？」 | 数値目標の設定 |
| 「いつまでに達成したいですか？」 | 期限の設定 |
| 「どんなライフスタイルを目指していますか？」 | ビジョンの把握 |
| 「学習に使える時間は週何時間ですか？」 | 現実的なプラン策定 |

## 6. キャリアパス提案ガイド

### 面談データから見える主要キャリアパス

34件の面談データ分析から、以下のパターンが最も多い:

#### パターンA: AIライター → コンサル（最多・最短マネタイズ）

**対象:** AI初心者、副業で早く稼ぎたい人、文章に抵抗がない人

```
Step 1: AIライティング習得（1〜2週間）
  └→ プロンプトが決まれば良い文章が書ける
Step 2: クラウドソーシングで案件応募（2〜4週間）
  └→ ランサーズ・クラウドワークス
Step 3: 実績を積む + SNSで発信
Step 4: AIコンサルタントへ展開
```

**実績データ:**
- 最短で月3万〜5万のマネタイズに到達した会員多数
- 面談34件中、AIライター推奨が最も多い

#### パターンB: AI画像/動画生成 → クリエイティブ職

**対象:** ビジュアルに興味がある人、Midjourney等を既に使っている人

```
Step 1: AI画像生成（Midjourney等）で毎日SNS投稿
  └→ ポートフォリオの蓄積
Step 2: 動画生成スキルの追加（画像→動画変換AI）
Step 3: クライアントへ提案（YouTube台本+動画セット）
```

**ポイント:**
- 趣味でも作品をSNSにアップし続けることが将来の実績になる
- 画像単体よりも動画にすることでクライアントへの訴求力UP

#### パターンC: 業務効率化 → 社内AI推進

**対象:** 本業がある会社員、独立は考えていない人

```
Step 1: 自身の業務でAI活用（議事録・リサーチ・文書作成）
Step 2: 削減時間を数値化（外注費換算でマネタイズ効果として報告）
Step 3: 社内でAI導入推進者として活躍
Step 4: 副業としてAIコンサル（任意）
```

**ポイント:**
- 「外注した場合の費用」を基準に削減額を計算 → マネタイズ効果
- Claude Code等のAIエージェント活用を推奨

#### パターンD: SNS運用 → マーケター

**対象:** X/Instagram等で既に発信している人、時間に制約がある人

```
Step 1: AIでSNSコンテンツ作成を効率化
Step 2: 自分のアカウントで成果を出す
Step 3: SNS運用代行案件の獲得
```

**ポイント:**
- 時間制限がない（クライアントワークのような納期プレッシャーが少ない）
- 本業の隙間時間でも進められる

### キャリアパス決定のフローチャート

```
Q: AIの経験はどのくらい？
├─ 初心者（ほぼ未経験）
│   └→ まずAIライターから（パターンA）
│       └→ 最短でマネタイズしたい → クラウドソーシング案件応募
│       └→ じっくり学びたい → SNS発信 + ポートフォリオ構築
│
├─ 少し触っている（ChatGPT程度）
│   ├─ 文章が好き → パターンA
│   ├─ ビジュアルに興味 → パターンB
│   └─ 業務効率化が目的 → パターンC
│
└─ ある程度使える
    ├─ 既にライティング実績あり → コンサル/マーケターへ展開
    ├─ 画像/動画生成経験あり → パターンB の Step 2〜
    └─ SNS運用経験あり → パターンD
```

## 7. プロプラン活用方法の説明ポイント

### 実践フィードバック会（最重要）

**必ず伝えること:**
- 毎日21:00〜22:00開催（日曜のみ11:00〜12:00）
- 少人数制で講師から直接フィードバックをもらえる
- リアルタイム参加が理想だが、アーカイブ視聴でもOK
- 出席レポートの提出が必要（全額返金保証の条件）

**曜日別テーマの案内:**

| 曜日 | テーマ | こんな人におすすめ |
|---|---|---|
| 月 | AIコンサル案件獲得実践会 | 案件獲得を目指す人 |
| 火 | 登壇練習会 | 講師・コンサルを目指す人 |
| 水 | コンサル現場ケーススタディ会 | 実践的な事例を学びたい人 |
| 木 | 案件応募・面談練習 | 応募・営業スキルを磨きたい人 |
| 金 | テスト案件添削 | 案件のクオリティを上げたい人 |
| 土 | 週間振り返り会 | 1週間の成果を整理したい人 |
| 日 | 特別会（月替わり）| 幅広く学びたい人 |

### Discord活用

- **プロ限定コミュニティ**: 意識の高い仲間との交流
- **何でも相談室**: 学習で詰まったらここで質問
- **日報集**: 毎日の活動記録を投稿

### 日報・週報

- **日報**: 毎日の学習・実践の記録。習慣化が最重要
- **週報**: 1週間の振り返り + マネタイズ成果の記入
- 提出内容は個人カルテに自動反映される
- 講師やコンシェルジュからのフィードバックに活用

### キャリア診断ボット

- 会員サイト「学習の歩き方」内にあるチャット形式ツール
- AI時代のキャリアルートを診断してくれる
- まだ方向性が決まっていない会員には積極的に案内

### 全額返金保証の条件

- 実践FB会への参加（リアル or アーカイブ）
- 出席レポートの提出
- 日報・週報の提出
- **面談に参加しないと返金対象外になる可能性がある** ことは、大笠さんに表現を確認してから伝えること

## 8. よくある会員パターンと対応例

### パターン1: インプット過多で動けない

**特徴:**
- コースをたくさん見ているが、実践に移せていない
- 「正解がわからない」「これで合っているのか不安」
- 完璧主義的な傾向

**対応:**
- 「完璧でなくていい。まず手を動かして、FB会でフィードバックをもらう」
- 「講師の方法を徹底的に模倣する。自己流は後から」
- 「毎日のFB会が修正の機会。毎日軌道修正できるから安心して進んで」
- 「今やらなくていいことはやらない。一つに絞る」

### パターン2: 何をしたいかわからない

**特徴:**
- AIに興味はあるが、具体的な方向性が定まっていない
- 「漠然とAIコンサルになりたい」

**対応:**
- キャリア診断ボットの利用を案内
- 最短マネタイズとしてAIライターを提案（ゼロイチが最も早い）
- 「やりながら見つかる。まず一つ始めてみましょう」

### パターン3: 多スキルだが収益化できていない

**特徴:**
- 動画編集、ライティング、デザインなど複数スキルを持つ
- しかしどれも収益につながっていない

**対応:**
- 「スキルは十分。足りないのはビジネスの導線」
- 一つに絞って案件獲得に集中することを提案
- プロフィール作成 → SNS発信 → クラウドソーシング応募の流れ

### パターン4: 時間が取れない

**特徴:**
- 本業が忙しい（シフト勤務、長時間労働等）
- 学習時間の確保が課題

**対応:**
- アーカイブ視聴の活用を強調
- 通勤時間・隙間時間でのインプットを提案
- 時間制限のない分野（SNS運用等）を推奨
- 「料金は支払済み。アーカイブでも必ず参加して元を取って」

### パターン5: 既に実績がある（ライティング案件獲得済み等）

**特徴:**
- 既にマネタイズ実績がある
- 次のステップに進みたい

**対応:**
- 実績を評価した上で、次の展開を提案
- AIコンサル、マーケター、ディレクター等へのステップアップ
- 「AI × 既存キャリア」の掛け合わせで差別化
- 単価アップの方法（複合スキル提供）

### パターン6: 業務効率化が主目的（副業意向なし）

**特徴:**
- 本業でのAI活用が主目的
- 副業・独立は考えていない

**対応:**
- 業務効率化の削減効果をマネタイズ効果として報告できることを説明
- Claude Code等のAIエージェント活用を推奨
- 社内でのAI推進者ポジションの価値を伝える

## 9. 面談で使えるフレーズ集

### 導入

- 「プロプランへの加入、ありがとうございます。今日はプロプランでの目標を一緒に整理していきましょう」
- 「事前アンケートの内容を拝見しましたが、もう少し詳しくお話を聞かせてください」

### 目標設定

- 「プロプランで達成したい目標を、金額と時期で教えてもらえますか？」
- 「その目標、具体的で達成不可能な数値ではないですね。一緒に達成しましょう」
- 「まず1ヶ月でここまで、3ヶ月でここまで、という段階を決めましょう」

### キャリアパス提案

- 「スピード重視で稼ぎたいなら、まずAIライターがおすすめです」
- 「プロンプトさえ決まれば良い文章が書けるので、ゼロイチが最も早いです」
- 「実績を積みながら、次のステップとしてコンサルや動画に展開していきましょう」
- 「AI × あなたの既存スキルを掛け合わせることで、すべてが仕事になります」

### アウトプット促進

- 「インプットを貯め込まず、毎日のFB会でアウトプットすることが大事です」
- 「講師のやり方を徹底的に模倣してください。自己流は後から」
- 「趣味でもSNSにアップし続けてください。それが将来の実績になります」
- 「完璧でなくていいです。毎日FB会で修正できるので、安心して進んでください」

### Discord活用

- 「Discordの何でも相談室を使ってください。一人で悩まなくて大丈夫です」
- 「プロのコミュニティには意識の高い仲間がいます。積極的に交流してください」
- 「2月（3月）の同期メンバーとのつながりを大切にしてください」

### クロージング

- 「今日決めたアクションをまとめると…」
- 「困ったことがあればいつでもDiscordで相談してください」
- 「一緒に頑張りましょう！」

## 10. 面談後の対応

### 必須タスク

- [ ] Geminiメモ（文字起こし）のURLをカルテの「初回面談文字起こし」列に記入
- [ ] 面談内容のまとめをカルテの「初回面談議事録」列に記入

- [ ] Discord・会員サイトのアクセス設定が完了していることを確認
- [ ] 必要に応じて次回面談の予約案内

### フォローアップ

- 面談から1週間後: 日報・FB会参加状況を確認
- 参加がない場合: 優しい確認メッセージを送信
- オンボーディング進捗をチェックリストで管理
