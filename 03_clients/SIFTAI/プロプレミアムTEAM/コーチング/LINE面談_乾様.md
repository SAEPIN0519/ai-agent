# LINE面談テンプレート｜乾様

> 対応理由: 聴覚障害のため、通常のGoogle Meet面談（音声30分）の代わりにLINEチャットで実施
> 面談種類: PRO初回面談（正規入会者向け）
> 担当: 野村冴香

---

## 面談の進め方（LINE版）

- 質問は**1〜2個ずつ**送る（一度に大量に送らない）
- 相手の返信を待ってから次へ進む
- テキストなので相手のペースに合わせる（即レスを求めない）
- 面談記録はLINEのトーク履歴がそのまま記録になる
- 所要時間の目安は決めず、何日かに分けてもOK

---

## メッセージテンプレート

### ① 導入（Phase 1）

```
乾さん、はじめまして！
SHIFT AI PROプランの面談を担当させていただく野村冴香です😊

通常はGoogle Meetでお話しするのですが、
LINEチャットで進めさせていただきますね！

今日の面談の目的は3つです👇
・プロプランでの目標を一緒に整理する
・今の状況を確認する
・目標に合ったキャリアプランを考える

ゆっくりで大丈夫なので、
一つずつお答えいただけると嬉しいです🙌

早速ですが、最初の質問です！
```

### ② 入会背景のヒアリング（Phase 2）

```
【質問1】
AIに興味を持ったきっかけを教えていただけますか？
また、プロプランを選んだ理由もあれば教えてください😊
```

↓ 返信を待つ

```
【質問2】
現在のお仕事について教えてください！
・会社員 or フリーランス or その他
・お仕事の内容（差し支えない範囲で）
・副業として考えていますか？それとも独立を目指していますか？
```

### ③ 現状把握（Phase 3）

```
【質問3】
普段、AIツールはどのくらい使っていますか？

例えば…
・ChatGPT、Claude、Geminiなど → 使ったことある？
・Midjourney（画像生成）→ 興味ある？
・その他のAIツール

「全然使ったことない」でも全く問題ないです！
今のレベルに合わせてプランを考えますね👍
```

↓ 返信を待つ

```
【質問4】
会員サイトのコースや動画は、もう見始めていますか？

※ちなみに、動画教材に字幕がないものもあるかもしれません。
もし視聴しづらいコンテンツがあったら遠慮なく教えてくださいね。
テキストベースの学習方法も一緒に考えます！
```

### ④ 目標設定（Phase 4）

```
【質問5】
乾さんがプロプランで達成したい目標を教えてください！

ざっくりでOKです👇
・1ヶ月後にやっていたいこと
・3ヶ月後に達成したいこと
・1年後の理想の姿

金額目標がある方は「月○万円稼ぎたい」でもOKですし、
「まずはAIを使いこなせるようになりたい」でも大丈夫です😊
```

↓ 返信を待つ（ここが最重要。カルテのワーク④⑤を埋めるためのヒアリング）

```
【質問6】
10年後のイメージも聞いてみたいです！
・どんな働き方をしていたいですか？
・どんな場所で暮らしていたいですか？
・理想の年収やライフスタイルは？

答えにくければ「まだイメージできていない」でも全然OKです。
一緒に考えていきましょう！
```

### ⑤ キャリアパス提案（Phase 5）

> ※ここは相手の回答を踏まえてカスタマイズする。
> 乾さんのカルテ診断はAIライター方向だが、本人の意向を確認した上で提案。

**AIライター方向の場合:**
```
乾さんのお話を聞いて、AIライターからスタートするのがおすすめかなと思いました！

ステップはこんなイメージです👇

Step 1（1〜2週間）
→ AIライティングの基礎を学ぶ

Step 2（2〜4週間）
→ ブログやnoteで実績を作る

Step 3（1〜3ヶ月）
→ クラウドソーシングで案件に応募

Step 4（3ヶ月〜）
→ 実績をもとに単価UP・AIコンサルへ展開

乾さんは学習時間が日3時間確保できているので、
しっかり進められると思います💪

この方向性、いかがですか？
他に気になる分野があれば教えてくださいね！
```

**業務効率化方向の場合:**
```
乾さんのお話を聞いて、今のお仕事の業務効率化から始めるのが良さそうですね！

Step 1: まず自分の業務でAIを使ってみる（議事録・リサーチ・文書作成）
Step 2: 削減できた時間を数値化する
Step 3: 社内でAI活用の推進者になる
Step 4: 副業でAIコンサルも視野に

「外注したら○万円かかる作業をAIで自動化」
→ これも立派なマネタイズ効果として報告できます！

この方向性はいかがですか？
```

### ⑥ プロプラン活用方法（Phase 6）

```
【プロプランの活用方法をお伝えしますね！】

✅ 実践FB会（フィードバック会）
→ 毎週開催。学んだことをアウトプットして講師からフィードバックをもらえます
※ 音声中心のイベントなので、参加が難しい場合はアーカイブ＋テキスト質問で対応できます。詳しくは相談しましょう！

✅ Discord
→ 何でも相談室で質問OK。テキストベースなので乾さんも使いやすいと思います！

✅ 日報
→ 毎日の学習記録を投稿。習慣化が成長のカギです

✅ コンシェルジュ
→ AI活用の相談ができます。テキストで質問できるので安心してください
```

### ⑦ クロージング（Phase 7）

```
今日お話しした内容をまとめますね！

📌 目標:
（相手の回答を入れる）

📌 キャリアの方向性:
（提案した内容を入れる）

📌 まずやること:
（具体的な最初の一歩を入れる）

何か困ったことがあったら、いつでもLINEで聞いてくださいね！
Discordの何でも相談室もテキストで質問できるので活用してください😊

一緒に頑張りましょう！🙌
```
