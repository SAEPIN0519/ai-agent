# Slackからの情報検索・呼び出し

## 概要

Claude Codeから直接Slack APIを使って、チャンネルのメッセージを検索・取得するスキル。
「あの人が言ってたあれ、何だっけ？」をSlackを開かずにClaude Codeから解決できる。

## 使い方

Claude Codeに以下のように指示するだけ：

- 「Slackで青嶋さんがドライブについて言ってたの探して」
- 「Slackの直近1週間で〇〇について話してた内容を教えて」
- 「Slackから△△さんのメッセージを探して」

## 仕組み

### 必要なもの
- Slack Bot Token（`09_system/config/slack_bot_token.txt` に保存済み）
- `slack_sdk` Pythonパッケージ（インストール済み）
- チャンネルID（`C0AC8404FPE` = community_ss_pro-premium_plan）

### 検索パターン

#### 1. キーワード検索（特定の話題を探す）
```
Slackで「ドライブ」「共有」について話してた内容を探して
```
→ メッセージ本文をキーワードでフィルタリング

#### 2. 人物指定検索（誰が何を言ったか）
```
Slackで青嶋さんのメッセージを探して
```
→ ユーザー名でフィルタリング

#### 3. スレッド深掘り（議論の流れを追う）
```
Slackのスレッドも含めて探して
```
→ 親メッセージ＋スレッド返信をすべて取得

### 既存スクリプト
- `09_system/slack_reader.py` — チャンネル活動のサマリー取得用
  - `python slack_reader.py --days 7` で直近1週間
  - `python slack_reader.py --member "野村冴香"` で特定メンバー

## 活用シーン

| シーン | 指示例 |
|---|---|
| 誰かの発言を探す | 「Slackで岩間さんがエルメについて言ってたの探して」 |
| 共有URLを探す | 「Slackで共有されたGoogle DriveのURLを探して」 |
| 議論の経緯を確認 | 「Slackでオンボーディングについての議論を見せて」 |
| タスク依頼を確認 | 「Slackで青嶋さんからの依頼メッセージを探して」 |
| 報告書の材料集め | 「Slackの直近1ヶ月のメンバー活動をまとめて」 |

## 注意点

- Bot Tokenの権限範囲内のチャンネルのみ検索可能
- 大量取得する場合はAPI制限に注意（1回200件まで）
- DMは取得不可（チャンネルメッセージのみ）
