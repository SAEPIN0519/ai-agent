## SIFTAI Claude Code勉強会 第8回宿題

### Google Sites認証アクセス調査
- 宿題ギャラリー（sites.google.com/10xc.jp/claude-code）にプログラムからアクセスを試行
- 試行した方法と結果:
  - OAuth Bearer token → Google Sites非対応
  - Google Drive API → 新Google SitesはDrive非公開
  - Chrome Cookie直接読み取り → Chrome排他ロック
  - browser_cookie3 + Shadow Copy → App-Bound Encryption(v20)で復号不可
  - Selenium（同プロファイル） → Chrome実行中クラッシュ
  - **VSS（Volume Shadow Copy） + DPAPI復号 → ファイルコピー成功、ただしv20暗号化で復号不可**
- 結論: Chrome 127+のApp-Bound Encryptionにより外部Cookie取得は原理的に不可能
