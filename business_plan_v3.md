## 2. 不労所得化の4ステップ

### Step 1: 自分でやる（Month 1-3）
**目的: 「型」を作る**

| やること | 詳細 |
|---|---|
| モニター10件を自分たちで対応 | 実際にやることで「何が必要で、何が不要か」を体で覚える |
| 全工程を記録・マニュアル化 | 1件ごとに「手順書」を作りながら進める |
| テンプレート一式を完成させる | CLAUDE.md雛形、settings.json雛形、ガードレール、スキルパック |
| 導入チェックリストを作成 | パートナーが「これを上から順にやれば完了」の状態にする |
| 品質基準を定義 | 「ここまでやったらOK」の明確なライン |

**この3ヶ月が事業の全て。ここで作る「型」の精度が、不労所得の安定性を決める。**

---

### Step 2: パートナーに任せる（Month 4-6）
**目的: 「型」が他人でも再現できることを証明する**

| やること | 詳細 |
|---|---|
| パートナー5人を採用・育成 | SHIFT AI関連のAI活用に強い人材 |
| 認定プログラムを実施 | 型を使って自分たちの前でデモ導入を実施 → 合格で認定 |
| 最初の5件は同行 | パートナーの横について品質を確認 |
| 6件目から独り立ち | 同行なしで対応。報告だけもらう |
| 報告フォーマットを固める | パートナーが「何を報告するか」を標準化 |

**パートナー認定の条件:**
1. Claude Code導入支援を自分で1件完遂できる
2. チェックリスト通りに設定できる
3. クライアントのヒアリングから要件を引き出せる
4. セキュリティ設定を間違いなくできる

---

### Step 3: パートナー管理を仕組み化する（Month 7-9）
**目的: 「パートナーの管理」も自動化する**

| やること | 詳細 |
|---|---|
| 報告システムを構築 | パートナーが完了後にフォームで報告 → 自動通知 |
| 品質チェックを自動化 | 納品物のチェックリストを定型化。抜き打ち確認は月1回程度 |
| 集客を仕組み化 | SNS + LP + 紹介制度で、問い合わせが自動的に入る状態 |
| 案件アサインの仕組み化 | 問い合わせ → 地域/専門/空き状況でパートナーに自動アサイン |
| パートナー育成の仕組み化 | 新規パートナーは既存パートナーが育成（OJT） |

**ここで重要なのは「Aosyさんがやらなくても回る仕組み」を作ること。**

---

### Step 4: 報告を受けるだけ（Month 10〜）
**目的: 不労所得の完成**

| **合計稼働** | | **月3〜4時間** |

## 4. パートナーからの報告フロー

```
パートナーが導入支援を実施
    ↓
完了報告フォーム送信（自動）
  - クライアント名
  - 実施日
  - 設定した内容（チェックリスト）
  - BeforeAfter数値
  - クライアント満足度（5段階）
  - 特記事項・トラブル
    ↓
自動通知（Slack or Discord → Aosy + SAEKA）
    ↓
週1回まとめて確認（Aosyさんの唯一の作業）
    ↓
月末: SAEKAさんがPLレポート作成
    ↓
月1回パートナーMTG（オンライン1時間）
```

### 報告で見るべき指標（これだけ見ればOK）

| 指標 | 基準 | アクション |
|---|---|---|
| 月間件数 | 40件以上 | 下回ったら集客を強化 |
| クライアント満足度 | 平均4.0以上 | 3.5以下が続くパートナーは面談 |
| パートナー稼働率 | 各人月3〜5件 | 偏りがあればアサイン調整 |
| トラブル件数 | 月2件以下 | 3件以上なら原因分析 |
| 月次利益 | 300万以上 | 下回ったら価格 or 件数を見直し |

## 5. 不労所得を守る仕組み

### パートナーが独立するリスクへの対策

最大のリスは「パートナーが型を覚えて独立する」こと。対策:

| # | 対策 | 内容 |
|---|---|---|
| 1 | ブランドの力 | First Codeブランドで集客 → パートナー個人では案件が来ない状態にする |
| 2 | 集客の一元化 | 問い合わせはすべてFirst Code経由。パートナーに直接来ない |
| 3 | テンプレートの継続更新 | Claude Codeのアップデートに合わせてテンプレートを更新。最新版はFirst Code経由でしか手に入らない |
| 4 | コミュニティの力 | パートナー同士のナレッジ共有コミュニティ。独立するとこのネットワークを失う |
| 5 | 報酬の魅力 | 月4件 × 17万 = 月68万。副業として十分魅力的。独立するより楽 |

### Aosyさんが動けなくなった場合の保険

| シナリオ | 対策 |
|---|---|
| Aosyさんが1ヶ月不在 | SAEKAさんが報告確認を代行。仕組み化済みなので問題なし |
| Aosyさんが3ヶ月不在 | パートナーリーダー（最も実績のあるパートナー）に管理を委任 |
| Claude Code市場の縮小 | 「AI業務自動化支援」にピボット。パートナー網はそのまま使える |
