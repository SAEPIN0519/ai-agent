# AIエージェント設計：日報提出状況 監視・リマインドエージェント

## 役割
会員カルテから日報提出状況を読み取り、手が止まっている会員を段階的にフォローする。
3日連続未提出はエルメで全体アナウンス、5日連続停止は個別に15分面談を促進する。

## 担当業務
1. 会員カルテの日報提出履歴を毎日読み取り
2. 連続未提出日数を会員ごとに集計
3. 3日連続未提出 → エルメで対象会員へアナウンス送信
4. 5日連続未提出（手が止まっている） → エルメで15分面談の申し込みを促すメッセージ送信
5. 冴香さんへの週次状況レポート送付

---

## 判定ロジック

```
毎日チェック（会員カルテから読み取り）:

  連続未提出 3日目:
    → エルメ: 全体アナウンス or 個別メッセージ（内容：日報提出を促す）

  連続未提出 5日目（手が止まっている判定）:
    → エルメ: 15分面談の申し込みURLを送付
    → 冴香さんへアラート（要フォロー候補として共有）

  連続未提出 5日超:
    → 冴香さんへ「緊急フォロー」フラグとして週次レポートに記載
```

---

## エルメ送信メッセージ案

### 3日連続未提出（アナウンス）
```
【日報リマインド】

最近、日報の投稿が止まっているメンバーへ。

小さなことでもOKです。
「今日はここまでやった」の記録が、
明日のあなたの行動につながります。

一緒に前に進みましょう！
```

### 5日連続未提出（面談促進）
```
【サポートのご案内】

ここ数日、活動が止まっているのが気になっています。

うまくいかない理由があるなら、
一緒に整理しませんか？

15分のオンライン面談を無料でご用意しています。
気軽に話しかけてください。

▼ 面談申し込みはこちら
[申し込みURL]
```
