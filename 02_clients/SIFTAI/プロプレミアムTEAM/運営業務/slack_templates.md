# Slack 定型文集

用途別にまとめた定型文。コピペして使用 or AIエージェントに自動送信させる。

---

## 1. 日報回収促進

### 日報リマインド（夜）
```
おつかれさまです！
本日の日報、まだの方はぜひ提出してください。
小さなアクションでも大丈夫です。
継続が力になります。
→ 日報・週報チャンネル: [URL]
```

### 日報リマインド（翌朝）
```
おはようございます！
昨日の日報がまだの方は、今朝でもOKです。
振り返ってから今日をスタートしましょう。
```

---

## 2. 週報回収促進

### 週報リマインド（日曜昼）
```
今週も1週間おつかれさまでした！
週報の提出はお済みですか？
今週の成果・気づき・来週の目標を記録しておきましょう。
→ 週報チャンネル: [URL]
```

### 週報リマインド（日曜夜）
```
今日中に週報の提出をお願いします！
1週間の振り返りが来週の質を上げます。
→ 週報チャンネル: [URL]
```

## 3. FB会参加促進

### 当日リマインド
```
今夜21:00〜 実践FB会があります！
テーマ：[テーマ名]
ぜひリアルタイムで参加してください。
都合が悪い方はアーカイブ視聴で参加できます。
```

### アーカイブ促進
```
昨日のFB会のアーカイブが公開されています。
見逃した方はこちらから→ [アーカイブURL]
視聴後は出席レポートの提出もお忘れなく！
→ 出席レポート: [フォームURL]
```

## 4. 非アクティブ会員への個別フォロー

### 優しい確認メッセージ（1〜2週間音沙汰なし）
```
[名前]さん、最近どうですか？
日報や週報が見られなくて少し心配しています。
うまくいっていないことや困っていることがあれば、
いつでも個別でお話しましょう！
```

### もう少し直接的なフォロー（3週間以上）
```
[名前]さん、お元気ですか？
PRO活動が少し止まっているように見えて気になっています。
一度現状を整理する時間を取りませんか？
30分ほど話せる日時を教えてください。
```

## 5. 表彰・称賛

### アクティブ会員への称賛
```
今週も積極的に活動してくれてありがとうございます！
[名前]さんの[具体的な行動]が素晴らしかったです。
この調子で来週も一緒に頑張りましょう！
```

### Lv.アップ報告への反応
```
[名前]さん、Lv.[N]達成おめでとうございます！
[具体的な成果]を達成した努力、本当にすごいです。
次のステージも全力でサポートします！
```

## 6. オンボーディング対応

### 加入直後ウェルカム（PREMIUM）
```
[名前]さん、PREMIUM加入ありがとうございます！
専属コンシェルジュの[冴香]です。
まず初回面談の日程を調整させてください。
ご都合の良い時間帯を教えていただけますか？
```

### 初回面談前リマインド
```
[名前]さん、明日[日時]にPRO初回面談がありますね。
事前に初回アンケートのご記入をお願いします。
→ [アンケートURL]
当日よりスムーズに話が進みます！
```
