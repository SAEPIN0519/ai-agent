## 講座の目的

> 「うちの店には、いつ・何をすればお客さんが来るのか」を90分で明確にする

- 無料AIツール「FELO」と一緒に、**自分の店に合った集客のやり方**を見つける
- ネット（SNS）とリアル（店頭・口コミ）の両方で、**明日からできるアクション**を持ち帰る
- 参加者同士でシェアし合い、**他の人の工夫からもヒントをもらう**

## 各ワークの詳細

### ワーク① 自店の現状を書き出す（10分）

アクションシートの上段に記入：

**【基本情報】**

| 記入項目 | 例 |
|----------|-----|
| 業種 | カフェ、美容室、整体院 など |
| 営業日・営業時間 | 月〜土 10:00-18:00 |
| 忙しい曜日・時間帯 | 土曜の午前、平日ランチタイム |
| 暇な曜日・時間帯 | 火曜終日、平日15時台 |
| 一番の悩み | 平日の午後にお客さんが来ない |

**【今やっている集客の棚卸し】**

| 項目 | 記入欄 | 例 |
|------|--------|-----|
| 使っているSNS | | Instagram、LINE公式、なし |
| Instagramの投稿頻度 | | 週1回、月2〜3回、ほぼ放置 |
| Instagramで発信しているキーワード・内容 | | 新メニュー紹介、店内の雰囲気、スタッフ紹介 |
| SNS以外でやっている集客（リアル） | | チラシ、口コミ、看板、地域イベント |
| うまくいっていると感じること | | 口コミで来る人は多い |
| うまくいっていないと感じること | | Instagramを見て来たという人がいない |

### ワーク② 講師と一緒にFELOで自店を分析する（30分）

**進め方：講師がスクリーンで実演 → 参加者も同時に自分のスマホ/PCで実行**
**プロンプト詳細：** → [集客ロードマップ_FELOプロンプト集.md](集客ロードマップ_FELOプロンプト集.md)

#### ステップ1（8分）：お客様の「1日」を見える化する ＋ 今のSNSを診断
- 講師がFELOを開いて実演 →「では一緒に開いてください」
- 配布プロンプトの〇〇を自店の情報に置き換えて入力（**今やっているSNS・キーワードも入力**）
- 回答から**「隙間時間」「Instagramの診断結果」「おすすめハッシュタグ」**をシートにメモ

#### ステップ2（10分）：「知る→気になる→行ってみる」の流れを分析
- 同じチャット画面のまま、ステップ2のプロンプトを入力
- 回答の表から**「最適な媒体」**をシートにメモ
- 「最後の一言」で一番しっくり来るものに丸をつける

#### ステップ3（12分）：明日からできるアクションプランを作る（ネット/リアル別）
- 同じチャット画面のまま、ステップ3のプロンプトを入力
- **ネット施策3つ・リアル施策3つ**をシートに書き写す
- それぞれ「明日やること」に★をつける

※ 時間に余裕がある参加者は**ボーナス ステップ4（優先順位整理）**も使用可

## グループシェア（10分）

> 杉山館長方針：**参加者同士でシェアし合い、互いの気づきを持ち帰る**

**進め方（6人全員で）：**

1. 全員のアクションシートをテーブルに並べる（または回し読み）
2. 1人1分で**自分のシートのポイントを口頭で共有**：
   - 「自分の店は今〇〇をやっていて、FELOで調べたら〇〇がわかった」
   - 「明日から〇〇をやろうと思います」
3. 聞いている人は**お客さん目線でひとことフィードバック**：
   - 「その投稿、〇〇時に見たら行きたくなる」
   - 「リアルでこういう工夫してるのは参考になる」
4. **他の人のシートから「これ自分もやりたい」を1つ盗む** → 自分のシートに追記OK

## 講座運営メモ

- **FELOの操作説明を個別にやらない**。講師がスクリーンで見せながら「一緒にやりましょう」で進める
- プロンプトは紙で配布し、**そのまま入力すれば動く状態**にしておく
- 各質問は**講師が先に実演 → 参加者が自分の業種に置き換えて入力**の流れで統一
- データを持っていない参加者は**体感値でOK**。業種別データシートも活用
- 発表に抵抗がある参加者には**ペア共有だけでもOK**と伝える
- アシスタント1名いると理想的（FELOの操作サポート用）
