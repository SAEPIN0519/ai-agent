# 週次ジャーナル振り返り

## 実行フロー

1. **ジャーナル読み込み**: `03_journal/YYYY-MM/` から直近7日分のジャーナルファイルを読み込む
2. **作業サマリ抽出**: 各日の実施内容・決定事項を時系列で整理
3. **未完了タスク集約**: 「次回への申し送り」セクションから未着手・進行中のタスクを抽出
4. **週次サマリ出力**: 以下のフォーマットで出力

## 出力フォーマット

```markdown
# 週次振り返り（YYYY/MM/DD〜MM/DD）

## 今週やったこと
- （日付ごとに箇条書き）

## 決定事項
- （重要な意思決定を列挙）

## 持ち越しタスク
- [ ] （未完了タスクをチェックリスト形式で）

## 来週の優先事項（提案）
- （申し送り内容から優先度順に提案）
```
