# 会員日報ウィークリー分析レポート

## 実行フロー

1. **日報データ収集**: `02_clients/SIFTAI/` 配下の会員カルテから直近1週間分の日報を読み込む
2. **要点抽出・感情分析**: 各日報から要点を抽出し、ポジティブ／ネガティブの発言比率を算出。具体的な声もそのまま集計
3. **今週のアクティブTOP3抽出**: 提出率の高い会員TOP3を特定し、1人あたりの稼働平均時間を算出
4. **非アクティブ会員の検出**: 提出件数3件以下の会員の名前・Discord名を抽出
5. **週次レポート出力**: 以下のフォーマットで出力

## 出力フォーマット

```markdown
# 週次日報分析レポート（YYYY/MM/DD〜MM/DD）

## 全体サマリ
- 提出率: XX%
- アクティブ会員: X名 / 全X名

## アクティブTOP3
| 順位 | 会員名 | 提出率 | 平均稼働 | 今週の声 |

## 要フォロー会員
| 会員名 | Discord名 | 提出数 | 備考 |

## 心境変化ピックアップ
- （ポジティブ・ネガティブの変化が大きい会員を原文引用付きで記載）
```

## 注意事項

- 数値だけで判断せず、生の声の原文引用を必ずセットで出力する
